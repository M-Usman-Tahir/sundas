Information Management System

Description:

IMS or Information Management System is actually a Student Database Management Program where all the Studentâ€™s Data can easily be stored and retrieved. It helps to get rid of traditional File system and enter into a virtual world. IMS can be used at all levels for a school, college, academy or any educational institute. IMS has three major portals, one for Administrators, one for Students and one for teachers..


Features of IMS:

1. Separate logins for Administrators, Teachers and Students.

2. User-friendly, attractive and colorful Graphical Interface. It comprises of proper Icons, Menus and Buttons with Instructions and Messages indicating actions.

3. Timely Updates and Notifications on Portals as well on Emails.

4. Sending Emails to all Students and all Teachers on One Click.

5. Remote Access.


Tools Used:

1. Python is used for writing the Basic codes. PyCharm Environment is used here.

2. Flask Web Framework is used to support the development of web application.

3. Jinja Templates to replace variables in templates with actual values at runtime.

4. HTML to create webpages and CSS to set the presentation and layouts of webpages.

5. MySQL Database to store data into and retrieve data from. XAMPP Control Panel is used as an IDE for MySQL Database.
